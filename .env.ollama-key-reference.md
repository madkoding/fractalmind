# Ollama Cloud API Key
# 
# To use Ollama Cloud instead of local Ollama:
# 1. Get your API key from https://ollama.com/
# 2. Copy it and create a .env file with:
#    OLLAMA_API_KEY=your_actual_api_key_here
#    OLLAMA_BASE_URL=https://api.ollama.com
# 3. Update EMBEDDING_PROVIDER=ollama, CHAT_PROVIDER=ollama, etc.
#
# Note: The real API key should be stored in .env (not committed to git)
# Example key format: xxxxxxxxxxxxxxxxxxxxxxxx.xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
